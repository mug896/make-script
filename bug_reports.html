Bug reports

책 내용 중에 수정, 보완해야 될 부분이 있거나 또는 추가하면 좋을 내용이 있으시면
아래 github issues 게시판을 통해 알려주시면 감사하겠습니다.

https://github.com/mug896/make-script/issues

make 4.3 버전의 deb 패키지가 아직 나오지 않았을 경우는 임시로 https://pkgs.org/download/make 사이트에서 rpm 이나 arch 패키지의 make 4.3 / x86_64 버전을 받아서 make 실행파일만 바꾸어 주면 됩니다.

https://mageia.pkgs.org/cauldron/mageia-core-release-x86_64/make-4.3-2.mga8.x86_64.rpm.html
위 사이트 에서 make-4.3-2.mga8.x86_64.rpm 파일을 다운로드

$ cd `mktemp -d` && rpm2cpio ~/path/to/make-4.3-2.mga8.x86_64.rpm | cpio -idv

$ sudo cp ./usr/bin/make /usr/bin/make
